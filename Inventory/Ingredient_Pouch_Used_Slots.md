# Ingredient_Pouch_Used_Slots function

## Aliases

- POUCH_USED

## Description

Returns the amount of used slots in your ingredient pouch.

## Arguments

This function has no arguments.

## Schematics

```js
INGREDIENT_POUCH_USED_SLOTS() → (Integer)
```

## Example

```js
INGREDIENT_POUCH_USED_SLOTS → 11
```
