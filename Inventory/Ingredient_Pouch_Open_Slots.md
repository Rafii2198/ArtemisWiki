# Ingredient_Pouch_Open_Slots function

## Aliases

- POUCH_OPEN
- POUCH_FREE

## Description

Returns the amount of free slots in your ingredient pouch.

## Arguments

This function has no arguments.

## Schematics

```js
INGREDIENT_POUCH_OPEN_SLOTS() → (Integer)
```

## Example

```js
INGREDIENT_POUCH_OPEN_SLOTS → 5
```
