# Current_World_Uptime function

## Aliases

- WORLD_UPTIME
- UPTIME

## Description

Returns the time since the world your are currently in restarted as String.

## Arguments

This function has no argument.

## Schematics

```js
CURRENT_WORLD_UPTIME() → (String)
```

## Example

```js
CURRENT_WORLD_UPTIME() → "2h 24m"
```
