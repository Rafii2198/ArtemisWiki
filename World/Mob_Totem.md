# Mob_Totem function

## Aliases

None

## Description

Returns the location of Mob Totem.

## Arguments

This function has 1 **required** argument.

- totemNumber(Integer) → The number of the mob totem to check.

## Schematics

```js
MOB_TOTEM(<totemNumber>) → (Location)
```

## Example

```js
MOB_TOTEM(1) → [-507, 64, -835]
```
