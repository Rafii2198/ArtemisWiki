# Min function

## Aliases

None

## Description

Returns the lowest value.

## Arguments

This function has 2 **required** arguments.

- first (Number) → The first number.
- second (Number) → The second number.

## Schematics

```js
MIN(<first>; <second>) → (Double)
```

## Example

```js
MIN(2000;3) → 3
```
