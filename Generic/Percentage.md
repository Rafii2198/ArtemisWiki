# Percentage function

## Aliases

- PCT

## Description

Returns the percentage of a CappedValue.

## Arguments

This function has 1 **required** argument.

- capped(CappedValue) → The capped value.

## Schematics

```js
PERCENTAGE(<capped>) → (Double)
```

## Example

```js
PERCENTAGE(capped_soul_point) → 75
```
