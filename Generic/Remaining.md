# Remaining function

## Aliases

- REM

## Description

Returns the difference of max and current value of a CappedValue.

## Arguments

This function has 1 **required** argument.

- capped(CappedValue) → The capped value.

## Schematics

```js
REMAINING(<capped>) → (Integer)
```

## Example

```js
REMAINING(capped_health) → 0
```
