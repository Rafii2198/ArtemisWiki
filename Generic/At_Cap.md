# At_Cap function

## Aliases

- AT_CAP

## Description

Returns true when the value reached the cap of CappedValue.

## Arguments

This function has 1 **required** argument.

- capped(CappedValue) → The capped value.

## Schematics

```js
AT_CAP(<capped>) → (Boolean)
```

## Example

```js
AT_CAP(capped(10;10)) → true
```
