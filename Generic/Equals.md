# Equals function

## Aliases

- EQ

## Description

Returns true when 2 numbers are the same

## Arguments

This function has 2 **required** arguments.

- first (Number) → The first number.
- second (Number) → The second number.

## Schematics

```js
EQUALS(<first>; <second>) → (Boolean)
```

## Example

```js
EQUALS(8;13) → FALSE
```
