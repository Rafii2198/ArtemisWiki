# Max function

## Aliases

None

## Description

Returns the highest value.

## Arguments

This function has 2 **required** arguments.

- first (Number) → The first number.
- second (Number) → The second number.

## Schematics

```js
MAX(<first>; <second>) → (Double)
```

## Example

```js
MAX(2000;3) → 2000
```
