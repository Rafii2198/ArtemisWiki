# Not_Equals function

## Aliases

- NEQ

## Description

Returns true when 2 numbers are not the same

## Arguments

This function has 2 **required** arguments.

- first (Number) → The first number.
- second (Number) → The second number.

## Schematics

```js
NOT_EQUALS(<first>; <second>) → (Boolean)
```

## Example

```js
NOT_EQUALS(1;2) → TRUE
```
