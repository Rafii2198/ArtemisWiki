# Weaponsmithing_Percentage function

## Aliases

- WEAPONSMITHING_PCT

## Description

Returns the percentage of your Weaponsmithing.

## Arguments

This function has no arguments.

## Schematics

```js
WEAPONSMITHING_PERCENTAGE() → (Double)
```

## Example

```js
WEAPONSMITHING_PERCENTAGE → 8.83
```
